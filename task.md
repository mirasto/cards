**ТЗ:** «Каталог контактів» (User Directory)
**Мета:** Створити міні-додаток для перегляду та пошуку користувачів.
**API:** Використовуй безкоштовне API Random User Generator.
**Endpoint:** https://randomuser.me/api/?results=20 (завантажить 20 випадкових юзерів).

1. Основні вимоги (MVP)
**Data Fetching:** При завантаженні сторінки стягнути дані з API.

Обов'язково обробити стани: Loading (поки вантажиться) та Error (якщо API впало).
**Rendering**: Відобразити список користувачів у вигляді карток (Card) або таблиці.
Дані для показу: Аватарка, Повне ім'я, Вік, Країна.


**Filtering:** Додати інпут.  При введенні тексту список має фільтруватися по Імені або Прізвищу (case-insensitive, тобто нечутливо до регістру).

**Sorting:** Додати перемикач (select або кнопки) для сортування:
- За алфавітом (A-Z).
- За віком (від молодих до старших).

2. Технічні вимоги (Stack)
React (Functional Components + Hooks).
TypeScript (опиши інтерфейс User).
Стилізація: CSS-modules, Tailwind або MUI (на твій вибір, знаючи що ти любиш MUI — можеш взяти його для швидкості).
